go-memcached
============

Package `memcached` provides an implementation of [httpcache.Cache](https://github.com/gregjones/httpcache/blob/master/httpcache.go#L30) that stores and retrieves data using a Memcache ([client](https://github.com/bradfitz/gomemcache)).

It's based on [mgo-cache](https://github.com/mcuadros/go-mgo-cache).
